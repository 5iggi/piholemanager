## Overview

Pi-hole Manager installs and manages Pi-hole natively on LoxBerry. The plugin provides a web interface for installation, web port handling, password management, health checks, dnsmasq.d files and DHCP/PXE checks.

## Design

Pi-hole Manager is a management layer around a native Pi-hole installation. Pi-hole remains a standalone system component using the normal Pi-hole paths under `/etc/pihole`, `/opt/pihole` and `/var/log/pihole`.
