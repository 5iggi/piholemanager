## Übersicht

Pi-hole Manager installiert und verwaltet Pi-hole nativ auf LoxBerry. Das Plugin stellt eine Weboberfläche für Installation, Web-Port, Passwortverwaltung, Healthchecks, dnsmasq.d-Dateien und DHCP/PXE-Prüfungen bereit.

## Arbeitsweise

Pi-hole Manager ist eine Verwaltungsschicht um eine native Pi-hole Installation. Pi-hole selbst bleibt eine eigenständige Systemkomponente mit den üblichen Pi-hole-Pfaden unter `/etc/pihole`, `/opt/pihole` und `/var/log/pihole`.
