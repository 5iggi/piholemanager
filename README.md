## 🇩🇪 Deutsch

Pi-hole Manager ist ein LoxBerry Plugin zur nativen Installation und Verwaltung von Pi-hole auf dem LoxBerry. Das Plugin stellt eine Weboberfläche für Installation, Port-Konfiguration, Passwortverwaltung, Healthchecks, dnsmasq.d-Konfigurationsdateien, DHCP/PXE-Erkennung und Backups bereit.

### Funktionen

- Native Pi-hole Installation auf LoxBerry
- Unterstützung für Pi-hole v6 und pihole-FTL
- Web-Port-Erkennung und Web-Port-Änderung
- Passwortfunktion für das Pi-hole Webinterface
- Healthcheck für Pi-hole, pihole-FTL, DNS und Port 53
- dnsmasq.d Editor für `.conf` Dateien
- DHCP/PXE-Erkennung mit Leasefile-Prüfung
- Schutz von System- und Plugin-Konfigurationsdateien
- Backups vor Konfigurationsänderungen
- LoxBerry 4 orientiertes Webinterface
- Autoupdate-Dateien `release.cfg` und `prerelease.cfg`

## 🇬🇧 English

Pi-hole Manager is a LoxBerry plugin to install and manage a native Pi-hole instance on LoxBerry. The plugin provides a web interface for installation, web port configuration, password handling, health checks, dnsmasq.d configuration files, DHCP/PXE detection and backups.

### Features

- Native Pi-hole installation on LoxBerry
- Pi-hole v6 and pihole-FTL support
- Web port detection and configuration
- Password handling for the Pi-hole web interface
- Healthcheck for Pi-hole, pihole-FTL, DNS and Port 53
- dnsmasq.d editor for `.conf` files
- DHCP/PXE detection with leasefile checks
- Protection for system and plugin configuration files
- Backups before configuration changes
- LoxBerry 4 oriented web interface
- Autoupdate files `release.cfg` and `prerelease.cfg`
